#Programa de Graficación 3D v 2.0

Este es un proyecto desarrollado en Java para la clase de Graficación, impartida por el Ing. José Lino Hernández Omaña.

El programa permite visualizar una figura en 3D y aplicarle las transformaciones geométricas básicas (traslación, escalación y rotaciones), tanto en su forma directa como inversa. También muestra el historial de coordenadas exactas para comprobar los cálculos.
